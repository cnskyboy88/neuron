## Getting Started

### 1st: git clone nerveCentre

	git clone git@f2e.dp:nervecentre.git
	cd path/to/nervecentre/
	npm link
	cd path/to/neuron/
	neuron start . -d

### 2nd: 访问主页

visit: http://localhost:1337
